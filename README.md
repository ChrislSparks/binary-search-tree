# Binary Search Tree

## Description
A Binary Search Tree (BST) implementation in C++ that demonstrates efficient data storage and retrieval using a hierarchical tree structure. This project includes recursive algorithms for insertion, deletion, searching, traversals, and tree analysis while emphasizing proper pointer management and dynamic memory allocation.

## Features
- Insert new nodes into the tree
- Delete nodes (leaf, one child, and two children)
- Search for existing values
- Inorder traversal (sorted output)
- Preorder traversal
- Postorder traversal
- Find minimum and maximum values
- Calculate tree height
- Recursive implementation of core operations
- Automatic memory cleanup using a destructor

## Concepts Covered
- Binary Search Trees (BST)
- Recursive algorithms
- Tree traversal techniques
- Dynamic memory allocation
- Pointer manipulation
- Object-oriented programming (OOP)
- Time complexity of tree operations

## Technologies
- C++
- Standard Template Library (STL)

## Time Complexity

| Operation | Average | Worst Case |
|----------|---------|------------|
| Search | O(log n) | O(n) |
| Insert | O(log n) | O(n) |
| Delete | O(log n) | O(n) |
| Traversal | O(n) | O(n) |
